Charter for the Enlightening — Repository Overview

This document serves as the entry point to the Formation Archive.

This repository contains the foundational materials for A Charter for the Enlightening.

The Charter is not a fixed system or doctrine.
It is a set of principles and an ongoing record of conversations exploring how individuals and societies can become more self-aware, less harmful, and more capable over time.
